# Marketing Intelligence

AI-powered marketing optimization platform - Your Fractional CMO.

## Features

- **Campaign Analysis**: Multi-dimensional scoring with performance tracking
- **Channel Optimization**: ROAS analysis, budget allocation, channel mix optimization
- **Content Strategy**: Content performance by funnel stage, engagement tracking
- **Funnel Analytics**: Conversion optimization, leak identification
- **ROI Analysis**: CAC/CLV tracking, marketing spend efficiency
- **Industry Benchmarks**: KPI comparison against industry standards
- **AI Consultation**: Claude-powered CMO consultant

## Tech Stack

- **Backend**: Flask, SQLAlchemy, PostgreSQL
- **AI**: Anthropic Claude API
- **Frontend**: Bootstrap 5
- **Deployment**: Render

## Quick Start

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY=your-key
python web/app.py
```

## AI Conversation Modes

- **General**: General marketing questions and guidance
- **Campaign Analysis**: Analyze campaign performance and optimize

- **Content Strategy**: Content marketing strategy and performance
- **Funnel Analysis**: Marketing funnel optimization
- **ROI Review**: Marketing ROI and spend efficiency
- **Benchmark Discussion**: Industry benchmarks and competitive analysis
- **Marketing Planning**: Strategic marketing planning and roadmaps
